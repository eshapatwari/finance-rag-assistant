# Finance Research Assistant

A Retrieval-Augmented Generation (RAG) application for analyzing financial documents such as annual reports, earnings transcripts, investor presentations, and research reports.

The system ingests PDF documents, generates semantic embeddings, stores them in a vector database, and retrieves relevant context to answer user questions. It supports querying across multiple uploaded documents and provides source attribution for generated responses.

## Features

* Upload and ingest one or more PDF documents
* Automatic document chunking and embedding generation
* Semantic search using vector similarity
* Multi-document
